- 功能：

- 開發：
    - LabView：影像辨識、機械學習、電子元件控制、程式整合
    - 機構設計及機械元件：伸展機構（推高機）、聯軸器
        - 加工方式：雷射切割、車床
        - 設計：solidworks
- GitHub：https://github.com/Nick0603/Robot_capstone
- 備註：評分機制
    - 規定1：與定點座標的位置誤差
        - 以 5 cm 為單位來扣分到扣沒有為止
    - 規定2：舉高CD的延伸高度
        - 超過70cm 為 滿分
    - 規定3：機器人的總重
        - 需再1500g 以內是必須的
            - 每少100 g 有額外加分
            - 基本材料 (主機、底板、輪子、電池)就已經900g了
